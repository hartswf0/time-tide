This operational closure means that interventions must be translated into each subsystem's native code to be effective.

    // System responds to both the scenario and interventions

      metrics: systemState.getPathologyMetrics(),
      interventionCount: activeInterventions.length
    });
  }

    baselinePathology,
    finalPathology: results[results.length-1].metrics,
    trajectory: results
  };
}
